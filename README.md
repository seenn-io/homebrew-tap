# Homebrew Tap for Seenn

Official Homebrew tap for [Seenn](https://seenn.io) CLI tools.

## Installation

```bash
brew tap seenn-io/tap
brew install seenn
```

## Usage

```bash
seenn login      # Authenticate with Google
seenn whoami     # Show account info
seenn keys       # Show API keys
seenn logout     # Remove credentials
```

## Update

```bash
brew upgrade seenn
```

## Uninstall

```bash
brew uninstall seenn
brew untap seenn-io/tap
```
